# Portfolio with CMS

This project implements a simple portfolio website with a back-stage UI for managing projects.

## Install

```
npm install
```

## Run

```
npm start
```

The site will be available at `http://localhost:3000`.

## Features

- View portfolio projects on the main page.
- Admin page at `/admin.html` allows adding, editing and deleting projects.
- Projects are stored in `data/projects.json`.
